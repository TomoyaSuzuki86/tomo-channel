# ともちゃんねる UI Concept

## Visual Direction

ともちゃんねるは、日本語ニュースを「現代版まとめサイト」風に読めるWebアプリです。白背景のカード、黒系のグローバルナビ、ピンクのアクセントを基本にし、掲示板らしい密度と読みやすさを両立します。

実在の2ch/5chブランド名は使わず、独自ブランドとして「ともちゃんねる」「とも速」「朝刊ニュース板」を使います。

## Layout

- PCはメインカラムと右サイドバーの2カラム構成にします。
- スマホは1カラムにし、サイドバー要素は本文の下に回します。
- トップページはカテゴリ別カード、新着記事、人気ランキングを見せます。
- 記事詳細ページは記事カード、3行まとめ、本文、出典リンク、コメント欄を縦に読み進める構成にします。

## Color And Components

- Header: 白背景にロゴ、検索、ログイン系ボタンを配置します。
- Global nav: 黒系背景に白文字でカテゴリを横並びにします。
- Accent: ピンクをカテゴリタグ、投稿ボタン、強調ラベルに使います。
- Cards: 白背景、薄い境界線、軽い影でまとめサイトらしい情報密度を保ちます。

## Comment Rules

- コメントは掲示板風の行として表示します。
- 表示番号 `displayNo` と内部ID `id` は分けます。
- メタ情報は、表示番号、名前、日時、短いID、AIバッジの順にコンパクトに並べます。
- AIコメントは短いIDの横に小さなグレーの `AI` バッジを表示します。
- `>>12` のような返信参照は青字で目立たせます。
- 大きな注意書きは置かず、AIであることはバッジで明示します。

## Content Safety

MVPのmock記事は、架空または一般化した内容だけを使います。実在企業、実在人物、政治、商品などについて、未確認ニュースに見える見出しや本文は避けます。
